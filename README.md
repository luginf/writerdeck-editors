
# Writerdeck-editors

some editors for a writerdeck:

- writerdeck.lua : lua version
- writerdeck.py  : python version

Based on https://github.com/lallero7/WriterdeckForCMD, itself based on https://www.reddit.com/user/shmimel/

Converted to lua with the help of LLM (Claude code)

Nano and micro are also great tools for simple writerdecks. Simple, yet powerful. Those simpler editors are some kind of proof-of-concept for alternatives or other usages. 


## Usage:

lua writerdeck.lua : writerdeck mode, let you create and restore files from ~/Documents/writerdeck/ folder.
lua writerdeck.lua file.txt : editor mode, let you edit a file from any folder.

Same usage for the python version.
